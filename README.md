This is a simple browser-based car game built using HTML, CSS, and JavaScript in a single file.

The player controls a car using keyboard arrow keys while the road continuously scrolls. The objective is to survive as long as possible and achieve a high score without crashing.

Features:

Interactive start screen

Player-controlled car movement

Smooth scrolling road animation

Real-time score counter

Boundary collision detection

Game Over display

Continuous animation using requestAnimationFrame

Controls:

⬅️ Left Arrow → Move left

➡️ Right Arrow → Move right

Click on the start screen and use arrow keys to play.
